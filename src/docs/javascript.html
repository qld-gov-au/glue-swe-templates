QG Parent width

Custom JS

There may be some instances where you have to set a floated or absolutely positioned element to the same width as it's parent. To do this, add data-parent-width=true to the child element.

<div id="parent-element">
    <div data-parent-width="true">
        Your floated div.
    </div>
</div>
