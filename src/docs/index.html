Getting started

For now, use node package directly:

$ npm install web-template-release

Comming soon...

The prefered method for using this template is to use the Web Project Boilerplate from Git, which uses Node.js, NPM, and Gulp. This allows you to build both Franchise websites and Agency templates while using our best practice development methods. Note: This is comming soon.

  1. Ensure you have a Git client and Node.js installed
  2. Fork the Web Project Boilerplate repo:
    https://github.com/qld-gov-au/web-boilerplate/
    For more information see how to fork a github repo
  3. Set up your with NPM project: $ npm install $ npm init
  4. Check out the Web Project Boilerplate readme for instructions
  5. Read the documentation here to understand how to use the template

Technology

To help with rapid development, the web template is now tightly integrated with Bootstrap 3. It also includes jQuery and a suite of custom Javascript features.

Compliance

Queensland Government Web Template (web template) is build to comply with the IS26 Guidelines.

When building websites using this template, ensure you are complying with the Web Content Accessibility Guidelines (WCAG) 2.0 (WCAG 2.0) guidelines as outlined in IS26. This means AA compliance on all WCAG guidelines except Guideline 1.2, which requires A level complaince.

This template has been built to comply with WCAG 2.0 AA, but it cannot ensure content you add complies.

Useful tools for compliance:

  • WCAG testing: Squiz Labs HTML Code Sniffer
