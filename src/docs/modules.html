Accordion

An accordion displays collapsible content panels for presenting information in a limited amount of space. An accordion helps users see only the content they need by expanding and collapsing sections of content.

Considerations have been made to ensure the behaviour of the accordion is predictable for users. For example, a downward facing chevron has been used to indicate the direction of movement when expanding content. Additionally, the user is given full control over what content is visible by allowing the expansion of multiple panels at any time.

Additional features for the accordion include the following options:

  • Addition of an expand all/collapse all feature to an accordion, which is especially useful to the user when there is a large amount of information.
  • Addition of an image or icon (from the Font Awesome icon set) in line with the heading. This is not advised best practice.
  • Addition of sub-text beneath the heading. This is not advised best practice. If this is required, it is recommended that this is limited to no more than 2 lines.
  • Linking directly to an open accordion panel.
